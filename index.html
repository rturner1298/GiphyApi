About

This web application uses the Giffy api to return a number a Gifs that attempt to match the expression you have presented to the app.

Start by selecting a rating. Click on a button that has the text of the expression you wish to view GIFs for! You can also enter your own unique expression in the textbox and select using the enter key. You can then select the button to view the related gifs from the internet.
